Purpose

These demonstrations are designed for law students with no technical background. Each illustrates a core AI concept — learning, opacity, and hallucination — using legally relevant examples and real AI systems.

Technology

Demo 1 runs entirely in your browser using Brain.js. Demos 2 and 3 use Claude (Anthropic) accessed through Puter.js — a free platform requiring no API key. Each user's own Puter account is used; no data is stored.

Usage

All demos run in a standard web browser on desktop or mobile. Demos 2 and 3 require a free Puter account — a sign-in prompt will appear automatically on first use. All files are static HTML; no server or installation is required.
